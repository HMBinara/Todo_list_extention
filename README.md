## üöÄ Installation

1. **Download/Clone** this repository to your local machine.
2. Open Chrome and navigate to `chrome://extensions/`.
3. Enable **Developer mode** (toggle at the top right).
4. Click **Load unpacked** and select the folder containing these files.
5. Voila! The task orbit will appear on every webpage you visit.

## üí° How to Use

- **Add Task:** Type your task in the input field and press **Enter**.
- **Prioritize:** The first task added is automatically assigned **Red** (Highest Priority).
- **Complete Task:** Click on the task text to mark it as completed. The corresponding ring segment will turn off.
- **Delete Task:** Use the `√ó` button to remove individual tasks.
- **Clear All:** Use the "Clear All Tasks" button to wipe the list instantly.
